Version 1.0#

  • Feature: something that you couldn’t do before.

  • Efficiency: an existing feature now may not require as much computation or memory.

  • Enhancement: a miscellaneous minor improvement.

  • Fix: something that previously didn’t work as documentated

  • API: you will need to change your code to have the same effect in the future; or a feature will be removed in the future.

Dependencies#

Wildboar 1.0 requires Python 3.6+, numpy 1.17.3+, scipy 1.3.2+ and scikit-learn 0.24+.

Version 1.0.4#

Changelog#

  • Feature Model agnostic counterfactual explanations has been added.

  • Feature Synthetic generation of outlier detection datasets.

  • Feature IsolationShapeletForest has been added. A novel semi-supervised method for detecting time series outliers.

  • Feature Fast computation of scaled and unscaled dynamic time warping (using the UCRSuite algorithm).

  • Feature LB_Keogh lower bound and envelope.

  • Enhancement Shapelet forest counterfactual explanations has been refined.

  • Enhancement KNearestNeighbors counterfactual explanations has been refined.

  • Enhancement Add new class Repository which represents a collection of bundles

  • Enhancement datasets.set_cache_dir to globally change the default cache directory

  • Enhancement datasets.clear_cache to clear the cache

  • Enhancement wildboar.datasets.install_repository now installs a repository instead of a bundle

  • API datasets.load_all_datasets has been replaced by load_datasets

  • API Rename Repository to Bundle
